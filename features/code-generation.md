## Description

Create automated tools that generate boilerplate, configuration, and scaffolding code for entities, products, and infrastructure. Reduces manual work and ensures consistency.

## Inputs

- Templates and generation rules
- Input parameters (entity name, config, etc)
- Canonical structure specs (Vesta)

## Outputs

- Generated code/config files
- Integration-ready output (no post-generation fixes needed)
- Documented generation command
- Ready for manual adjustment as needed

- **Entity scaffolding** - part of gestation workflow
