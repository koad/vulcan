**Type:** Tech Stack

## Description

Primary development platform for koad:io products and services. Includes collection design, publications/subscriptions, methods, and integrations with the Blaze template engine.

## Inputs

- Product/feature specifications
- Database schema requirements
- Real-time data requirements

## Outputs

- Meteor applications
- MongoDB collections and indexes
- Publications and subscriptions
- Server methods and validation

## Tech Stack Details

- **Framework:** Meteor (JavaScript full-stack)
- **Database:** MongoDB
- **UI:** Blaze (Meteor's templating engine)
- **Authentication:** Meteor built-in (customized)
- **Build:** Meteor CLI
