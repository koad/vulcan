# Folder Conventions

## Sites / Web Projects

When koad says "create a bare project in [folder]" or similar for new sites:

1. Create folder at `~/.vulcan/sites/[folder]`
2. Run `vulcan spawn bare` in that folder

## Other Projects

For non-site projects, determine the appropriate location based on context:
- Code projects → `~/dev/` or `~/projects/`
- Entities → `~/.vulcan/projects/`
- Tools → `~/.koad-io/`
