# Vulcan Documentation Index

This directory contains the documentation structure for Vulcan capabilities.

## Organization

- **Architecture/** - System design and architectural decisions
- **Protocols/** - Protocol specifications and implementations
- **Infrastructure/** - Hosting, deployment, and ops documentation
- **Development/** - Development guides and tooling
- **API/** - API reference and usage guides
- **Guides/** - How-to guides and tutorials

## Getting Started

Start with the feature files in `../../features/` to understand what Vulcan builds, then follow links to detailed documentation.

## Protocol Specs

All public protocols follow VESTA specifications. See `../../features/` and `koad/vesta` repository for protocol details.
